# Unity Scripts Collection

A small collection of ready-made Unity C# scripts for small projects and prototypes.  
This repository is a toolbox of reusable scripts you can copy and use anywhere.

## Conent

- All Scripts → [`ReadySystems/Assets/Scripts`](ReadySystems/Assets/Scripts)
  - All UI → [`ReadySystems/Assets/Scripts/UI`](ReadySystems/Assets/Scripts/UI)
    - ToolTips System → [`ReadySystems/Assets/Scripts/UI/ToolTips`](ReadySystems/Assets/Scripts/UI/ToolTipS)


## Notes
- You can copy any folder you need into another Unity project (keep `.meta` files when copying).
