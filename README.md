# Unity Scripts Collection

A small collection of ready-made Unity C# scripts for small projects and prototypes.  
This repository is a toolbox of reusable scripts you can copy and use anywhere.

## Conent

- Scripts root → [`ReadySystems/Assets/Scripts`](ReadySystems/Assets/Scripts)
- UI → [`ReadySystems/Assets/Scripts/UI`](ReadySystems/Assets/Scripts/UI)
- ToolTips → [`ReadySystems/Assets/Scripts/UI/ToolTips`](ReadySystems/Assets/Scripts/UI/ToolTipS)

## Notes

- Scripts are organized by categories inside `Assets/Scripts`.
- You can copy any folder you need into another Unity project (keep `.meta` files when copying).
